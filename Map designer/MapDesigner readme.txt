RNG Map Designer.

Число в тулбоксе задает размер карты NxN (кол-во ячеек). после изменения числа нажмите map->new.
Правая кнопка мыши - стирать.
tiles.png - файл, из которого загружаются шаблоны кисти 16x16.

Поддерживается работа с бинарными файлами. Формат данных: byte width, byte height, byte[] tiles.

Планируется добавить инструменты zoom in/out, undo.

Рисуйте на здоровье! Хорошие карты нам пригодятся!
